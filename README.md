# 🔐 Cyber Physical System - Attack Detection using XGBoost

## 📌 Project Overview
This project presents a Machine Learning-based Cyber Attack Detection System designed for Cyber Physical Systems. It uses the powerful XGBoost algorithm integrated with a Django web application to detect and classify potential cyber threats. The system enables real-time monitoring, user interaction, and visualization of attack detection results.

---

## 🚀 Tech Stack
- Backend: Django (Python)
- Machine Learning: XGBoost
- Database: SQLite
- Frontend: HTML, CSS
- Libraries: NumPy, Pandas, Scikit-learn

---

## 🔥 Key Features
- 👤 User Registration & Login System  
- 🛠 Service Provider (Admin) Dashboard  
- 🤖 Cyber Attack Detection using ML Model  
- 📊 Detection Accuracy & Performance Analysis  
- 📁 Dataset Upload & Result Processing  

---

## ▶️ How to Run the Project

Step 1: Clone Repository  
git clone https://github.com/Kanaparthy79Roselin/Thwart-Attack-Detection-using-XGBoost-Django-Project.git  
cd Thwart-Attack-Detection-using-XGBoost-Django-Project  

Step 2: Install Dependencies  
pip install -r requirements.txt  

Step 3: Run Server  
python manage.py runserver  

Step 4: Open in Browser  
http://127.0.0.1:8000/
