# Cyber Physical System - Attack Detection

## 🔥 Project Description
Machine Learning based Cyber Attack Detection system using Django and XGBoost.

## 🚀 Tech Stack
- Python
- Django
- XGBoost
- SQLite
- HTML/CSS

## 📊 Features
- User Registration/Login
- Service Provider Dashboard
- Attack Detection using ML
- Accuracy Reports

## ▶ How to Run
pip install -r requirements.txt
python manage.py runserver
